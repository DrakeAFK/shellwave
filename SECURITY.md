# Security Policy

## Supported Versions

ShellWave is currently in **Alpha**. There are no supported LTS versions yet.

Security fixes will target the latest commit on `main` until versioned releases are introduced.

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Use GitHub private vulnerability reporting if it is enabled for this repository.

ShellWave handles SSH connections, host metadata, session cookies, and local machine access. Security reports are taken seriously.

When reporting a vulnerability, please include:

- A clear description of the issue
- Steps to reproduce
- Impact or risk
- Suggested fix, if known

We will review reports and provide a patch as soon as possible.
